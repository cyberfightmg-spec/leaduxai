# LeaduxAI.id

AI-powered Pinterest publisher - создавайте и публикуйте пины в 1 клик

## 🚀 Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Откройте http://localhost:3000
```

## 🛠️ Технологии

- **Next.js 15** - React фреймворк
- **TypeScript** - Типизация
- **Tailwind CSS v4** - Стилизация
- **Three.js + React Three Fiber** - 3D графика
- **GSAP + ScrollTrigger** - Анимации
- **Framer Motion** - React анимации
- **Lenis** - Плавный скролл
- **html2canvas** - Экспорт в PNG

## 📱 Секции

1. **Hero** - 3D Pinterest pin с анимацией
2. **Demo** - Генератор пинов с canvas
3. **Steps** - 4 шага использования
4. **Features** - Возможности платформы
5. **Pricing** - Тарифные планы
6. **Footer** - Ссылки и подписка

## ✨ Особенности

- 🎨 Генератор пинов с AI
- 🌙 Темная/светлая тема
- 📱 Mobile-first дизайн
- ⚡ PWA ready
- 🎯 SEO оптимизирован
- 🖼️ Экспорт в PNG
